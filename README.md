# Сборка на основе Gulp от Yuliia Liubenka
---
### Начало
Скопируйте файлы в папку проекта:

- gulpfile.js; 
- package.json;
- .babelrc.json;
- .gitignore.

```
1. Открыть в програме VSC терминал.
2. Установить node модули: **npm install**.
3. Создать папки и файлы: **gulp struct**.
4. Запустить gulp: **gulp**.
---
### Файловая структура
Начнем с папки app.
**Сначала запускаеm команду gulp struct и папки с файлами создаются автоматически, а потом уже запускаеm команду gulp**
- `app/css` — папка, куда нужно складывать сторонний CSS код (например, normalize, hamburger.css и .тп)
- `app/fonts` — папка, куда складываются ttf шрифты. Gulp конвертирует их в woff и woff2, подключает используя миксин, а выставляет правильное начертание шрифта
- `app/img` — папка для изображений. Сжатие изображений происходит при запуске сборке, но так как работает пакет - `gulp-cache`, gulp реагирует только на новые изображения
- `app/pages` — папка для html файлов, которые обрабатываются с помощью `gulp-fileinclude`
